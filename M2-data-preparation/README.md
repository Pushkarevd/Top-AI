# Модуль 2. Подготовка данных

Директория содержит MD-файлы с описаниями контрольно-измерительных материалов (КИМ) модуля 2.

Модуль проверяет способность собирать данные из разнородных источников, проверять их корректность и реализовывать процессы консолидации, очистки и загрузки данных с использованием ETL и ELT, а также разрабатывать воспроизводимые программные компоненты интеграции и преобразования данных.

## Список КИМ

| КИМ | Форма контроля | Краткая аннотация | Индикатор |
| --- | -------------- | ----------------- | :-------: |
| [КИМ-2.1](kim-01-colloquium.md) | Коллоквиум | Обсуждение процессов консолидации данных, сравнение подходов ETL и ELT | BD-2.2 |
| [КИМ-2.2](kim-02-test.md) | Тестирование | Тестовые задания по интеграции данных, качеству данных и этапам подготовки | BD-2.2 |
| [КИМ-2.3](kim-03-lab-etl.md) | Лабораторная работа | Реализация ETL/ELT-процесса сбора, очистки и загрузки данных из разнородных источников | BD-2.2, PL-1.3 |
