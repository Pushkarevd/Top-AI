## Единая шкала оценивания

Каждое оценочное средство оценивается по стобалльной шкале. Принятая в КИМ шкала перевода:

| Сумма баллов | Оценка |
| :---: | --- |
| 85–100 | отлично |
| 70–84 | хорошо |
| 55–69 | удовлетворительно |
| 0–54 | неудовлетворительно |
