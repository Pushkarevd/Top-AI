# Документация дисциплины

В этой папке хранится документация верхнего уровня:

| Файл | Назначение |
| --- | --- |
| [rpd.md](rpd.md) | рабочая программа дисциплины |
| [grading-system.md](grading-system.md) | система итогового оценивания: веса, пороги, шкала перевода баллов |
| [quality-checklist.md](quality-checklist.md) | проверка готовности репозитория к публикации |
| [rpd.docx](rpd.docx) | исходный документ РПД в формате DOCX |
| [fos.docx](fos.docx) | исходный черновик ФОС в формате DOCX |

Публикуемой версией документов являются MD-файлы. DOCX-файлы сохраняются как исходники для сверки и оформления по требованиям образовательной организации.
