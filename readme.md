# Технологии хранения больших данных

Дисциплина посвящена принципам работы с большими данными и современным подходам к их хранению.

## Цель курса

Сформировать системное понимание работы с данными и развить навыки проектирования, создания и развития надёжных, масштабируемых и эффективных решений в области инженерии данных.
