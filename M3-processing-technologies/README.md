# Модуль 3. Технологии и инструменты обработки данных

Директория содержит MD-файлы с описаниями контрольно-измерительных материалов (КИМ) модуля 3.

Модуль проверяет способность обоснованно выбирать технологии пакетной, потоковой и распределенной обработки, развертывать распределенное хранилище, реализовывать обработку данных с использованием Hadoop, Spark и Trino, а также проектировать и отлаживать конвейеры обработки данных с оркестрацией, мониторингом, проверками качества и Data Lineage.

## Список КИМ

| КИМ | Форма контроля | Краткая аннотация | Индикатор |
| --- | -------------- | ----------------- | :-------: |
| [КИМ-3.1](kim-01-control-work.md) | Контрольная работа | Выбор и обоснование технологий обработки для заданного сценария, оценка надежности и производительности | BD-4.1, BD-4.3 |
| [КИМ-3.2](kim-02-lab-distributed-processing.md) | Лабораторная работа | Развертывание распределенного хранилища и формирование детального слоя данных средствами Hadoop, Spark и Trino (занятия 3.3, 3.4) | BD-4.1, PL-1.3 |
| [КИМ-3.3](kim-03-lab-airflow.md) | Лабораторная работа | Организация оркестрации ETL/ELT-процессов хранилища с помощью Apache Airflow (занятие 3.6) | BD-4.3, PL-1.3 |
| [КИМ-3.4](kim-04-lab-openmetadata.md) | Лабораторная работа | Формирование Data Lineage и проверок качества данных на базе OpenMetadata (занятие 3.7) | BD-4.3, PL-1.3 |
