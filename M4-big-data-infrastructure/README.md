# Модуль 4. Технологии организации инфраструктуры больших данных

Директория содержит MD-файлы с описаниями контрольно-измерительных материалов (КИМ) модуля 4.

Модуль проверяет способность выбирать компоненты единого технологического стека, проектировать масштабируемую архитектуру инфраструктуры больших данных и обосновывать принятые решения.

## Список КИМ

| КИМ | Форма контроля | Краткая аннотация | Индикатор |
| --- | -------------- | ----------------- | :-------: |
| [КИМ-4.1](kim-01-test.md) | Тестирование | Тестовые задания по компонентам единого технологического стека | BD-5.1 |
| [КИМ-4.2](kim-02-lab-infrastructure.md) | Лабораторная работа | Проектирование прототипа масштабируемой инфраструктуры больших данных с обоснованием выбора компонентов | BD-5.1 |
| [КИМ-4.3](kim-03-survey.md) | Опрос | Проверка понимания базовых принципов построения, расширения и управления аппаратными и виртуальными ресурсами для систем больших данных, а также навыков осуществления выбора направления вспомогательных технологических решений для формирования единого стека работы с большими данными. | BD-5.1 |

Перечень вопросов, выносимых на промежуточную аттестацию, приведен в разделе [«Экзамен»](../Exam/README.md).
