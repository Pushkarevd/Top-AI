# Модуль 1. Модели хранения данных

Директория содержит MD-файлы с описаниями контрольно-измерительных материалов (КИМ) модуля 1.

Модуль проверяет способность сравнивать OLTP- и OLAP-системы, выбирать архитектуру хранения, проектировать многомерную модель, создавать объекты реляционного хранилища и анализировать планы SQL-запросов.

## Список КИМ

| КИМ | Форма контроля | Краткая аннотация | Индикатор |
| --- | -------------- | ----------------- | :-------: |
| [КИМ-1.1](kim-01-test.md) | Тестирование | Тестовые задания по моделям хранения и многомерному моделированию данных | BD-3.1 |
| [КИМ-1.2](kim-02-lab-storage-design.md) | Лабораторная работа | Проектирование архитектуры хранения: сравнительный анализ OLTP- и OLAP-систем, анализ плана запроса | BD-3.1 |
