### SQL-скрипты

#### 6. **scripts/01_create_oltp_schema.sql**
Создание нормализованной схемы OLTP:
- 8 таблиц (users, addresses, categories, products, orders, order_items, payments)
- Индексы для оптимизации
- Проверочные запросы

#### 7. **scripts/02_seed_oltp_data.sql**
Генерация тестовых данных:
- 1000 пользователей
- 10 категорий
- 500 товаров
- 10000 заказов
- 30000 позиций заказов
- Платежи для заказов

#### 8. **scripts/03_create_olap_schema.sql**
Создание схемы хранилища данных:
- Таблица фактов fact_sales
- 4 измерения (dim_date, dim_user, dim_product, dim_order_status)
- Индексы для ускорения запросов

#### 9. **scripts/04_etl_to_olap.sql**
ETL-процесс заполнения хранилища:
- Заполнение измерения времени
- Загрузка пользователей из OLTP
- Загрузка товаров из OLTP
- Загрузка фактов продаж
- Создание представлений для удобства анализа

## 🔧 Технические требования

### Минимальные требования

- **PostgreSQL:** 14+
- **ОЗУ:** 4 ГБ (рекомендуется 8 ГБ)
- **Диск:** 1 ГБ свободного места
- **ОС:** Windows 10+, macOS 10.15+, Linux

### Рекомендуемые настройки PostgreSQL

```conf
# postgresql.conf
shared_buffers = 256MB
work_mem = 16MB
maintenance_work_mem = 128MB
effective_cache_size = 512MB
```

## 🚀 Запуск с Docker (рекомендуется)

```bash
# 1. Запуск PostgreSQL
docker run --name postgres-lab \
  -e POSTGRES_PASSWORD=postgres \
  -p 5432:5432 \
  -d postgres:15

# 2. Проверка
docker ps

# 3. Создание БД
docker exec -it postgres-lab createdb -U postgres ecommerce_oltp
docker exec -it postgres-lab createdb -U postgres ecommerce_olap

# 4. Копирование скриптов в контейнер
docker cp labs/scripts/ postgres-lab:/scripts/

# 5. Выполнение скриптов
docker exec -it postgres-lab psql -U postgres -d ecommerce_oltp -f /scripts/01_create_oltp_schema.sql
docker exec -it postgres-lab psql -U postgres -d ecommerce_oltp -f /scripts/02_seed_oltp_data.sql
docker exec -it postgres-lab psql -U postgres -d ecommerce_olap -f /scripts/03_create_olap_schema.sql
docker exec -it postgres-lab psql -U postgres -d ecommerce_olap -f /scripts/04_etl_to_olap.sql

# 6. Подключение для работы
docker exec -it postgres-lab psql -U postgres -d ecommerce_oltp
```
